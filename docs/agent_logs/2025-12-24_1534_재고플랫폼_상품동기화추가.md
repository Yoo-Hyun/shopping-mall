# 재고 플랫폼 PRD - 상품 동기화 섹션 추가

> 수정일: 2025-12-24 15:34

## 추가된 내용: 8.8 상품 동기화

### 8.8.1 상품 Push (재고 → 외부 채널)
- 재고 플랫폼에서 등록한 상품을 야후/메이크샵 등에 자동 등록
- 필드 매핑 엔진: 플랫폼별 CSV 형식에 맞게 변환
- 동기화 방식: 실시간 API, CSV 내보내기, 예약 동기화

### 8.8.2 외부 주문 수신 프로세스
- 외부 채널 주문 자동 감지 (Webhook/폴링)
- SKU 매핑으로 내부 상품 식별
- 자동 재고 차감 + 다른 채널에 재고 업데이트 Push
- FEFO 적용: 상미기간 짧은 로트부터 차감

### 8.8.3 상품 동기화 API
- 필드 매핑: /api/mappings/:platform
- 상품 Push: /api/sync/products/push
- CSV: /api/export, /api/import
- 외부 주문: /api/orders/external

### 8.8.4 동기화 대시보드
- 플랫폼별 동기화 상태, 실패 내역, 주문 수신 현황

### 데이터 모델 추가
- PlatformFieldMapping: 플랫폼별 필드 매핑
- ExternalOrder: 외부 주문 수신 데이터
