# 재고 플랫폼 PRD - 보안 요구사항 섹션 추가

> 수정일: 2025-12-25 10:50

## 추가된 내용: 10. 보안 요구사항

### 10.1 인증 (Authentication)
- JWT (Access + Refresh Token)
- 로그인 시도 제한 (5회 실패 → 15분 잠금)
- 2FA 선택적 지원

### 10.2 권한 관리
- 플랫폼 관리자, 테넌트 관리자, 매니저, 일반 직원, API 전용
- 역할별 권한 정의

### 10.3 데이터 보안
- 비밀번호 bcrypt 해싱
- API 키 AES-256 암호화
- HTTPS (TLS 1.2+)

### 10.4 API 보안
- 속도 제한 (100 req/분)
- IP 화이트리스트
- CORS, XSS/SQL Injection 방지

### 10.5 감사 로그
- 로그인, 재고 변경, 상품 변경, 설정 변경, API 키, 동기화

### 10.6 세션 관리
- 동시 로그인 3개, 24시간 비활동 만료

### 10.7 보안 알림
- 새 기기 로그인, 비밀번호 변경, API 키 발급, 비정상 접근
