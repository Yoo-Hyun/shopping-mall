# 재고 플랫폼 PRD - 통합 주문/배송 관리 섹션 추가

> 수정일: 2025-12-25 11:41

## 추가된 내용: 4.9 통합 주문/배송 관리

### 문제 해결
- 여러 플랫폼 주문을 한 곳에서 관리
- 놓치는 주문 방지

### 주문 상태 흐름
- 외부 주문 수신 → 대기중 → 포장중 → 발송완료 → 배송완료

### 통합 주문 목록
- 전체/미처리/발송대기 필터
- 플랫폼별 표시 (야후, 메이크샵, 자체몰)
- 미처리 건수 알림

### 송장 출력 기능
- 개별/일괄 출력
- 택배사 연동 (야마토, 사가와, 우체국)
- 송장 번호 자동 등록

### 주문/배송 알림
- 미처리 주문 24시간 이상 대기 시 알림
- 발송 지연 알림

### API
- /api/orders: 주문 CRUD
- /api/orders/:id/ship: 발송 처리
- /api/orders/bulk-print: 일괄 송장 출력
- /api/orders/summary: 현황 요약
