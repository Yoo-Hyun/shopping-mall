# PRD 업데이트 - 재고 관리 시스템 추가

> 작성일: 2025-12-22 15:51

## 추가된 내용

### 9.1 재고 관리 시스템 (Phase 1)

**A. 온라인 쇼핑몰 재고 관리**
- 상품별 재고 수량 (stock 필드)
- 주문 시 자동 재고 차감
- 주문 취소/환불 시 재고 복구
- 품절 처리 및 저재고 알림

**B. 오프라인 매장 재고 관리**
- 매장(Store) 데이터 모델 추가
- 매장별 개별 재고 관리
- 오프라인 판매 기록
- 매장 간 재고 이동

**C. 통합 재고 관리**
- 온라인 + 오프라인 전체 재고 조회
- 실시간 재고 동기화
- 재고 변동 이력 추적
- 통합 재고 대시보드

## 예상 데이터 모델
- Store (매장)
- Inventory (재고)
- InventoryLog (재고 이력)
