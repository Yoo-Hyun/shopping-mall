# 재고 플랫폼 PRD - 입출고 프로세스 섹션 추가

> 수정일: 2025-12-23 14:36

## 추가된 내용: 9. 입출고 프로세스

### 입고 프로세스
- 바코드 스캔 → 상품 데이터 로드  
- 입고일 자동, 상미기간/수량/창고위치 입력
- 로트(Lot) 번호 자동 생성

### 출고 프로세스
- FEFO + FIFO 자동 선택
- 상미기간 짧은 것 → 먼저 입고된 것 순
- 부분 출고 지원

### 데이터 모델
- InventoryLot: 로트별 입고 데이터 관리

### API 추가
- /api/receiving (입고)
- /api/shipping (출고)
- /api/lots (로트 관리)

### 알림 기능
- 상미기간 7일/3일 전 알림
- 상미기간 경과 알림
- 저재고 알림
