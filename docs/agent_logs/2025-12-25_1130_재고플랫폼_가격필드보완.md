# 재고 플랫폼 PRD - 가격 필드 보완

> 수정일: 2025-12-25 11:30

## 수정된 내용: 공통 필드 (Core Fields)

### 추가된 필드
| 필드 | 타입 | 설명 |
|------|------|------|
| **정가 (희망소매가)** | Number | 정상 판매가격 (할인 전) |
| **할인율** | Number | 자동 계산: (정가-판매가)/정가×100 |

### 할인 표시 예시
```
정가: 1,000엔 / 판매가: 800엔 → 20% OFF 표시
```

### 참고
- 야후쇼핑 상품등록 화면에서 "희망소매가/정가" 필드 참고
- 할인율 자동 계산으로 각 판매 채널에서 할인 표시 가능
