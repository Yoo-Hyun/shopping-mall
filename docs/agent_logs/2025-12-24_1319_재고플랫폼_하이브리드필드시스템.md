# 재고 플랫폼 PRD - 하이브리드 필드 관리 시스템 추가

> 수정일: 2025-12-24 13:19

## 추가된 내용: 하이브리드 필드 관리 시스템

### 설계 방식
- 공통 필드 + 카테고리 템플릿 + 테넌트 커스터마이징

### 구성 요소

1. **공통 필드 (Core Fields)**
   - SKU, 바코드, 상품명, 가격, 이미지 등
   - 모든 상품에 필수

2. **카테고리 템플릿**
   - 식품: 유통기한, 알레르기, 영양성분
   - 화장품: 전성분, 사용기한, PAO
   - 의류: 사이즈, 컬러, 소재
   - 커스텀: 테넌트 직접 정의

3. **테넌트별 설정**
   - 필드 활성화/비활성화
   - 필드 순서 변경
   - 커스텀 필드 추가
   - 필드 필수/선택 설정

### 데이터 모델
- TenantFieldConfig: 테넌트별 필드 설정 관리

### API
- GET/PUT /api/field-config: 필드 설정 조회/저장
- POST /api/field-config/custom: 커스텀 필드 추가
- GET /api/templates: 템플릿 목록
