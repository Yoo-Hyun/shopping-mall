# 재고 플랫폼 PRD - 동기화 안전장치 섹션 추가

> 수정일: 2025-12-24 15:44

## 추가된 내용: 8.8.5 동기화 안전장치 (Safeguards)

### 동기화 모드 구분
- 재고만 동기화 (기본값 ON) - 가장 안전
- 상품 정보 업데이트 (기본 OFF)
- 신규 상품 등록 (기본 OFF)
- 상품 삭제 (기본 OFF) - 가장 위험

### 삭제 방지 규칙
- 삭제 기본 비활성화
- 10% 이상 삭제 시 수동 확인 필요
- 빈 데이터 Push 차단

### 동기화 미리보기
- 실행 전 변경 내용 확인 화면
- 재고 업데이트, 정보 변경, 신규 추가, 삭제 예정 표시

### 유효성 검사
- 빈 데이터, 필수 필드 누락, 대량 삭제, 가격 0원, 재고 음수 검사

### 백업 & 롤백
- 동기화 전 자동 백업
- 문제 시 롤백 가능
- 30일간 이력 보관

### API
- GET/PUT /api/sync/safeguards
- POST /api/sync/preview/:platform
- GET /api/sync/history
- POST /api/sync/rollback/:id
